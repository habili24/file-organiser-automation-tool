# file-organiser-automation-tool
A Python automation script that organizes files into folders based on file type, demonstrating problem-solving, workflow automation, and file system operations.
## File Organizer Automation Tool

This script automatically organizes files in a directory into folders based on file type.

### Why this matters
Manual file organization is time-consuming and error-prone. This script demonstrates how automation can improve efficiency.

### What it demonstrates
- File system operations
- Automation logic
- Clean control flow
- Practical problem-solving

### Limitations
- File types are hard-coded
- Does not handle subfolders
